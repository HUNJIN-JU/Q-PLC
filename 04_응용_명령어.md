# Q시리즈 응용 명령어

> 출처: Q대응 프로그래밍 매뉴얼(공통명령편-2001)

---

## 1. 데이터 전송 명령

| 명령 | 기능 | 문법 | 설명 |
|------|------|------|------|
| MOV | 16비트 전송 | MOV S D | S의 값을 D로 복사 |
| DMOV | 32비트 전송 | DMOV S D | 32비트 데이터 전송 |
| EMOV | 실수 전송 | EMOV S D | 부동소수점 전송 |
| BMOV | 블록 전송 | BMOV S D n | S부터 n워드를 D로 전송 |
| FMOV | 채움 전송 | FMOV S D n | D부터 n워드를 S값으로 채움 |
| SWAP | 상하위 교환 | SWAP D | 상위/하위 바이트 교환 |
| XCH | 교환 | XCH D1 D2 | D1↔D2 값 교환 |

### MOV 사용 예
```
LD    SM400
MOV   K100 D0    ; D0에 100 저장
MOV   D0  D10    ; D0 값을 D10으로 복사
DMOV  K100000 D20 ; D20~D21에 100000 저장(32비트)
```

## 2. 사칙 연산

### 16비트 연산

| 명령 | 기능 | 문법 | 설명 |
|------|------|------|------|
| + (P) | 덧셈 | + S1 S2 D | D = S1 + S2 |
| - (P) | 뺄셈 | - S1 S2 D | D = S1 - S2 |
| * (P) | 곱셈 | * S1 S2 D | D = S1 × S2 (결과 32비트) |
| / (P) | 나눗셈 | / S1 S2 D | D = S1 ÷ S2 (나머지: D+1) |

### 32비트 연산
- D+, D-, D*, D/ → 접두어 D 추가

### 연산 예
```
LD    SM400
+     D0 D2 D10     ; D10 = D0 + D2
-     D10 K5 D20    ; D20 = D10 - 5
*     D0 K10 D30    ; D30~D31 = D0 × 10
/     D0 K3 D40     ; D40 = D0 ÷ 3, D41 = 나머지
```

## 3. 비교 명령

| 명령 | 기능 | 설명 |
|------|------|------|
| LD= | 같으면 ON | S1 = S2 |
| LD<> | 다르면 ON | S1 ≠ S2 |
| LD> | 크면 ON | S1 > S2 |
| LD< | 작으면 ON | S1 < S2 |
| LD>= | 이상이면 ON | S1 ≥ S2 |
| LD<= | 이하이면 ON | S1 ≤ S2 |

```
LD>=  D0 K100
SET   M10         ; D0 ≥ 100이면 M10 ON
```

## 4. 논리 연산

| 명령 | 기능 | 설명 |
|------|------|------|
| WAND | AND | 비트 AND |
| WOR | OR | 비트 OR |
| WXOR | XOR | 배타적 OR |
| BSET | 비트 SET | 특정 비트 ON |
| BRST | 비트 RST | 특정 비트 OFF |
| BTEST | 비트 테스트 | 특정 비트 읽기 |

## 5. 변환 명령

| 명령 | 기능 | 설명 |
|------|------|------|
| BIN | BCD→BIN 변환 | 10진수→2진수 |
| BCD | BIN→BCD 변환 | 2진수→10진수 |
| INT | 실수→정수 변환 | 소수점 이하 버림 |
| FLT | 정수→실수 변환 | 부동소수점 변환 |
| GRY | BIN→그레이코드 | 그레이코드 변환 |
| GBIN | 그레이코드→BIN | 역변환 |

---

## 6. 프로그램 분기

| 명령 | 기능 | 설명 |
|------|------|------|
| CJ | 조건 점프 | 조건 ON 시 지정 포인터로 점프 |
| SCJ | 조건 분기 | 분기 후 복귀 없음 |
| CALL | 서브루틴 호출 | 포인터로 분기 후 RET으로 복귀 |
| RET | 서브루틴 복귀 | CALL 다음 행으로 복귀 |
| FOR | 반복 시작 | 지정 횟수만큼 반복 |
| NEXT | 반복 끝 | FOR와 쌍 |

```
LD    X0
CJ    P10       ; X0=ON이면 P10으로 점프

OUT   Y0        ; X0=OFF면 실행

P10             ; 점프 도착점
OUT   Y1
```
