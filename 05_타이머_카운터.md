# Q시리즈 타이머 & 카운터

> 출처: Q대응 프로그래밍 매뉴얼(공통명령편)

---

## 1. 타이머 (T)

### 타이머 종류

| 종류 | 시간 단위 | 범위 | 설명 |
|------|-----------|------|------|
| 저속 타이머 | 100ms | T0~T2047 | 기본 타이머 |
| 고속 타이머 | 10ms | 파라미터 설정 | 정밀 타이밍 |
| 적산 타이머 (ST) | 100ms | ST0~ST2047 | 정전 보상 |

### 사용법
```
LD    X0
OUT   T0 K50     ; X0=ON 동안 카운트, 5초(50×100ms) 후 T0 접점 ON

LD    T0
OUT   Y0          ; T0 타임업 시 Y0 출력
```

### 타이머 동작 원리
1. 입력 조건 ON → 현재값 카운트 시작 (0부터 증가)
2. 현재값 ≥ 설정값 → 타이머 접점 ON
3. 입력 조건 OFF → 현재값 0으로 리셋, 접점 OFF
4. **적산 타이머**: 입력 OFF 시 현재값 유지 (누적)

### 플리커 회로 (점멸)
```
LDI   T1
OUT   T0 K10     ; 1초 ON

LD    T0
OUT   T1 K10     ; 1초 OFF

LD    T0
ANI   T1
OUT   Y0          ; Y0 1초 ON/OFF 반복
```

## 2. 카운터 (C)

### 카운터 종류

| 종류 | 범위 | 카운트 | 설명 |
|------|------|--------|------|
| 16비트 카운터 | C0~C1023 | 1~32767 | 업카운트만 |

### 사용법
```
LD    X0
OUT   C0 K10     ; X0의 OFF→ON 횟수 카운트, 10회 시 C0 ON

LD    X1
RST   C0          ; C0 리셋 (현재값 0, 접점 OFF)
```

### 카운터 동작 원리
1. 입력 신호의 **상승 에지**(OFF→ON)를 카운트
2. 현재값 = 설정값 → 카운터 접점 ON (유지)
3. RST 명령으로 리셋해야 접점 OFF 및 현재값 0

### 활용 예: 생산 수량 카운트
```
; 센서 입력(X0)으로 제품 카운트
LD    X0
OUT   C10 K100    ; 100개 카운트

; 100개 도달 시
LD    C10
OUT   Y10         ; 만수 경보 출력
SET   M100        ; 만수 플래그

; 리셋
LD    X10
RST   C10         ; 카운터 리셋
RST   M100        ; 플래그 리셋
```

## 3. 고속 카운터 (QD62 모듈)

### QD62 개요
- **QD62**: 고속카운터 모듈 (2채널)
- **QD62D**: 고속카운터 모듈 (2채널, 차동 입력)
- **최대 카운트 속도**: 200kpps (1상), 100kpps (2상)
- **카운트 범위**: -2,147,483,648 ~ +2,147,483,647 (32비트)

### 카운트 모드

| 모드 | 입력 | 설명 |
|------|------|------|
| 1상 1카운트 | A상만 | 상승 에지에서 카운트 |
| 1상 2카운트 | A상만 | 상승+하강 에지 카운트 |
| 2상 1카운트 | A상+B상 | 4체배 없이 카운트 |
| 2상 4카운트 | A상+B상 | 4체배 카운트 (최고 분해능) |

### 일치 출력 기능
- **프리셋**: 일치 시 카운트값을 특정 값으로 변경
- **비교 출력**: 설정값 일치 시 외부 출력(COUT) ON/OFF
- **링 카운터**: 지정 범위에서 순환 카운트

### 버퍼 메모리 주요 주소

| 주소 | 기능 |
|------|------|
| Un\G0 | CH1 현재값 (하위) |
| Un\G1 | CH1 현재값 (상위) |
| Un\G2 | CH2 현재값 (하위) |
| Un\G3 | CH2 현재값 (상위) |
| Un\G19 | 카운트 모드 설정 |
| Un\G20 | 카운트 시작/정지 |
