# Q시리즈 기본 시퀀스 명령어

> 출처: Q대응 프로그래밍 매뉴얼(공통명령편-2001)

---

## 1. 접점 명령어

| 명령 | 기능 | 래더 기호 | 설명 |
|------|------|-----------|------|
| LD | a접점 시작 | ─┤ ├─ | 모선에서 시작하는 NO 접점 |
| LDI | b접점 시작 | ─┤/├─ | 모선에서 시작하는 NC 접점 |
| AND | a접점 직렬 | ─┤ ├─ | 직렬 연결 NO 접점 |
| ANI | b접점 직렬 | ─┤/├─ | 직렬 연결 NC 접점 |
| OR | a접점 병렬 | ─┤ ├─ | 병렬 연결 NO 접점 |
| ORI | b접점 병렬 | ─┤/├─ | 병렬 연결 NC 접점 |

### 펄스 검출 명령

| 명령 | 기능 | 설명 |
|------|------|------|
| LDP | 상승 에지 | OFF→ON 전환 시 1스캔 ON |
| LDF | 하강 에지 | ON→OFF 전환 시 1스캔 ON |
| ANDP | 상승 에지 직렬 | AND + 상승 에지 |
| ANDF | 하강 에지 직렬 | AND + 하강 에지 |
| ORP | 상승 에지 병렬 | OR + 상승 에지 |
| ORF | 하강 에지 병렬 | OR + 하강 에지 |

---

## 2. 출력 명령어

| 명령 | 기능 | 설명 |
|------|------|------|
| OUT | 코일 출력 | 조건 ON → 코일 ON |
| SET | 자기유지 ON | 한 번 ON 되면 계속 유지 |
| RST | 자기유지 OFF | SET된 코일 해제 |
| PLS | 상승 펄스 | 1스캔만 ON (OFF→ON 시) |
| PLF | 하강 펄스 | 1스캔만 ON (ON→OFF 시) |

### SET/RST 사용 예
```
LD    X0
SET   M0      ; X0이 ON되면 M0 자기유지

LD    X1
RST   M0      ; X1이 ON되면 M0 해제
```

---

## 3. 블록 접속 명령

| 명령 | 기능 | 설명 |
|------|------|------|
| ANB | 블록 직렬 접속 | 병렬 블록 후 직렬 연결 |
| ORB | 블록 병렬 접속 | 직렬 블록 후 병렬 연결 |
| MPS | 스택 저장 | 분기점 저장 |
| MRD | 스택 읽기 | 분기점 재사용 |
| MPP | 스택 꺼내기 | 마지막 분기점 사용 후 삭제 |

### MPS/MRD/MPP 사용 예
```
LD    X0
MPS           ; 분기점 저장
AND   X1
OUT   Y0      ; X0 AND X1 → Y0

MRD           ; X0 조건 재사용
AND   X2
OUT   Y1      ; X0 AND X2 → Y1

MPP           ; X0 조건 사용 후 삭제
AND   X3
OUT   Y2      ; X0 AND X3 → Y2
```

## 4. 마스터 컨트롤

| 명령 | 기능 | 설명 |
|------|------|------|
| MC N_ | 마스터 컨트롤 시작 | 조건 OFF 시 구간 내 출력 전체 OFF |
| MCR N_ | 마스터 컨트롤 해제 | MC 구간 종료 |

```
LD    X0
MC    N0 M100  ; X0=ON이면 아래 실행, OFF면 전체 OFF

LD    X1
OUT   Y0

LD    X2
OUT   Y1

MCR   N0       ; 마스터 컨트롤 해제
```

## 5. 종료 명령

| 명령 | 기능 | 설명 |
|------|------|------|
| END | 프로그램 끝 | 1스캔 종료, 선두로 복귀 |
| FEND | 메인 프로그램 끝 | 메인 루틴 종료 (서브루틴 전) |
| STOP | CPU 정지 | 강제 정지 (RUN→STOP) |
