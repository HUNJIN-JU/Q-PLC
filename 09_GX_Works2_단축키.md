## 1. 주요 단축키

### 파일/프로젝트

| 단축키 | 기능 |
|--------|------|
| Ctrl + N | 새 프로젝트 |
| Ctrl + O | 프로젝트 열기 |
| Ctrl + S | 저장 |
| Ctrl + Shift + S | 다른 이름으로 저장 |

### 편집

| 단축키 | 기능 |
|--------|------|
| Ctrl + Z | 실행 취소 |
| Ctrl + Y | 다시 실행 |
| Ctrl + X | 잘라내기 |
| Ctrl + C | 복사 |
| Ctrl + V | 붙여넣기 |
| Delete | 삭제 |
| Ctrl + F | 찾기 |
| Ctrl + H | 바꾸기 |

### 래더 편집

| 단축키 | 기능 |
|--------|------|
| F5 | a접점 (NO) |
| Shift + F5 | a접점 세로선 |
| F6 | b접점 (NC) |
| Shift + F6 | b접점 세로선 |
| F7 | 코일 출력 |
| Shift + F7 | 응용 명령 입력 |
| F8 | 수평선 |
| Shift + F8 | 수직선 |
| F9 | 수평선 삭제 |
| Shift + F9 | 수직선 삭제 |
| F4 | 변환 (래더→실행코드) |
| Ctrl + F4 | 전체 변환 |

### 온라인

| 단축키 | 기능 |
|--------|------|
| F3 | 모니터 시작/정지 |
| Shift + F3 | 모니터(쓰기 모드) |
| Ctrl + F3 | 현재값 변경 |

---

## 2. 온라인 조작

### PLC 읽기/쓰기

| 메뉴 | 기능 | 설명 |
|------|------|------|
| 온라인 > PLC 읽기 | PC ← PLC | PLC에서 프로그램 읽기 |
| 온라인 > PLC 쓰기 | PC → PLC | PLC로 프로그램 전송 |
| 온라인 > 리모트 조작 | RUN/STOP | 원격 운전/정지 |

### 모니터링 기능

| 기능 | 설명 |
|------|------|
| 래더 모니터 | 래더 위에서 ON/OFF 상태, 현재값 확인 |
| 디바이스 일괄 모니터 | 지정 범위 디바이스 값 표시 |
| 엔트리 모니터 | 특정 디바이스만 등록하여 감시 |
| 버퍼 메모리 모니터 | 특수 모듈 버퍼 확인 |

## 3. 디버깅 기능

### 강제 ON/OFF
- **Shift + Enter**: 선택 접점 강제 ON
- **Shift + Delete**: 선택 접점 강제 OFF
- 디바이스 테스트 창에서도 가능

### 현재값 변경
1. 모니터 모드에서 디바이스 더블클릭
2. 새 값 입력 > Enter
3. 또는 **Ctrl+F3** > 디바이스/값 입력

### 스텝 실행
- 1스텝씩 실행하며 디버깅
- **디버그 > 스텝 실행** 메뉴

---

## 4. 유용한 기능

### 크로스 레퍼런스
- 특정 디바이스가 사용된 **모든 위치** 검색
- **검색 > 크로스 레퍼런스** (Ctrl+Shift+F)

### 디바이스 코멘트
- 디바이스에 설명 텍스트 부여
- 래더 위에 코멘트 표시 → 가독성 향상
- **표시 > 코멘트 표시/숨기기**

### 프로그램 비교
- 두 프로그램 간 차이점 비교
- **도구 > 프로그램 비교**
