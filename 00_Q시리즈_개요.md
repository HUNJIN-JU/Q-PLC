# 미쓰비시 MELSEC-Q 시리즈 PLC 개요

> 출처: Q시리즈 데이터북, Q03UDECPU 매뉴얼, PLC기초교육자료

---

## 1. Q시리즈 시스템 개요

### MELSEC-Q 시리즈란?
- 미쓰비시전기의 **중·대형 PLC 플랫폼**
- 멀티CPU 시스템, 고속 처리, 대용량 프로그램 지원
- 다양한 네트워크(CC-Link, Ethernet, MELSECNET 등)와 통합

### 주요 라인업

| 카테고리 | 모델 예시 | 특징 |
|----------|-----------|------|
| 베이직 | Q00JCPU, Q00CPU | 소규모, 저비용 |
| 하이퍼포먼스 | Q03UDCPU, Q06UDCPU | 고속 연산, USB 내장 |
| 유니버설 | Q03UDECPU, Q06UDECPU | Ethernet 내장, 고기능 |
| 프로세스 | Q12PHCPU | 프로세스 제어 특화 |
| 이중화 | Q12PRHCPU | 이중화 시스템 |

### Q03UDECPU 주요 사양

| 항목 | 사양 |
|------|------|
| 프로그램 용량 | 30K 스텝 |
| 처리 속도 | LD 명령 34ns |
| 디바이스 점수 | I/O 최대 4096점 |
| 내장 Ethernet | 1포트 (100BASE-TX) |
| 내장 USB | 1포트 (프로그래밍용) |
| 최대 I/O 점수 | 4096점 |
| 프로그램 수 | 최대 124개 |

## 2. 시스템 구성

### 기본 구성 요소

```
[전원 유니트] + [CPU 유니트] + [I/O 유니트] + [특수 유니트]
     ↓              ↓             ↓              ↓
   Q61P          Q03UDECPU     QX42/QY42      QD62 등
```

### 베이스 유니트 종류

| 베이스 유니트 | 슬롯 수 | 용도 |
|--------------|---------|------|
| Q33B | 3슬롯 | 소규모 |
| Q35B | 5슬롯 | 소~중규모 |
| Q38B | 8슬롯 | 중규모 |
| Q312B | 12슬롯 | 대규모 |

### 증설 규칙
- **메인 베이스**: CPU가 장착되는 베이스 (1대)
- **증설 베이스**: 증설 케이블로 연결 (최대 7대)
- **증설 케이블**: QC05B(0.5m) ~ QC30B(3.0m)
- **전체 I/O 점수**: 베이스 유니트 장착 위치에 따라 자동 할당

## 3. 전원 유니트

### 주요 전원 유니트

| 모델 | 입력 전압 | DC5V 출력 | DC24V 서비스 전원 |
|------|----------|-----------|------------------|
| Q61P | AC100~240V | 6A | 없음 |
| Q62P | AC100~240V | 6A | 600mA |
| Q63P | DC24V | 6A | 없음 |

### 전원 용량 계산
- 각 유니트의 **DC5V 소비 전류**를 합산
- 전원 유니트 공급 용량을 **초과하지 않도록** 확인
- 부족 시 증설 베이스에 별도 전원 유니트 추가

## 4. 프로그래밍 환경

### GX Works2
- Q시리즈의 **표준 프로그래밍 소프트웨어**
- 래더(LD), SFC, ST, FBD 언어 지원
- 시뮬레이션, 모니터링, 디버깅 기능

### 주요 기능

| 기능 | 단축키 | 설명 |
|------|--------|------|
| 읽기(PLC→PC) | - | PLC에서 프로그램 읽기 |
| 쓰기(PC→PLC) | - | PLC로 프로그램 전송 |
| 모니터 시작 | F3 | 온라인 모니터링 |
| 래더 입력 | F5 | 접점(a접점) 입력 |
| 코일 입력 | F7 | 출력(코일) 입력 |
| 변환 | F4 | 래더 → 실행 코드 변환 |

### 접속 방법
1. **USB 접속**: Q03UDECPU 내장 USB 포트 → PC 직접 연결
2. **Ethernet 접속**: 내장 Ethernet 또는 QJ71E71-100 모듈
3. **시리얼 접속**: QC30R2 케이블 → RS-232C
