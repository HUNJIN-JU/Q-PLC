## 1. 내장 Ethernet 개요

### Q03UDECPU 내장 Ethernet 사양

| 항목 | 사양 |
|------|------|
| 포트 수 | 1포트 |
| 통신 속도 | 100Mbps (100BASE-TX) |
| 커넥터 | RJ45 |
| 프로토콜 | TCP/IP, UDP/IP |
| 동시 접속 수 | 최대 16커넥션 |
| 오토 네고시에이션 | 지원 |

## 2. IP 주소 설정

### GX Works2에서 설정
1. **프로젝트** > **파라미터** > **PLC 파라미터**
2. **내장 Ethernet 포트 설정** 탭 선택
3. IP 주소, 서브넷 마스크, 게이트웨이 입력

### 설정 예시

| 항목 | 값 |
|------|------|
| IP 주소 | 192.168.0.10 |
| 서브넷 마스크 | 255.255.255.0 |
| 기본 게이트웨이 | 192.168.0.1 |

### 주의사항
- IP 주소 변경 후 **전원 재투입** 또는 **리셋** 필요
- 같은 네트워크 내 IP 주소 **중복 불가**

## 3. 커넥션 설정

### 커넥션 종류

| 종류 | 용도 | 설명 |
|------|------|------|
| GX Works2 접속 | 프로그래밍 | PC에서 프로그래밍/모니터링 |
| MC 프로토콜 | 외부 기기 | SCADA/HMI 등에서 데이터 읽기/쓰기 |
| SLMP | 외부 기기 | 심레스 메시지 프로토콜 |
| 소켓 통신 | 범용 | TCP/UDP 직접 통신 |

### MC 프로토콜 (MELSEC Communication)
- 외부 기기(PC, HMI 등)에서 PLC 디바이스를 **읽기/쓰기**
- 바이너리 코드 / ASCII 코드 선택 가능

### MC 프로토콜 커맨드 예

| 커맨드 | 기능 |
|--------|------|
| 0401 | 일괄 읽기 (비트) |
| 0401 | 일괄 읽기 (워드) |
| 1401 | 일괄 쓰기 (비트) |
| 1401 | 일괄 쓰기 (워드) |
| 0619 | 리모트 RUN |
| 1619 | 리모트 STOP |

## 4. GX Works2 Ethernet 접속

### 접속 절차
1. PC와 PLC를 같은 네트워크에 연결
2. GX Works2 > **접속 대상** > **Connection Setting**
3. **Ethernet** 선택 > PLC IP 주소 입력
4. **통신 테스트** 클릭으로 확인

### 방화벽 설정
- Windows 방화벽에서 GX Works2 허용 필요
- 사용 포트: TCP 5001~5010 (기본)

## 5. 소켓 통신

### 소켓 통신 개요
- TCP/UDP를 사용한 **범용 데이터 통신**
- PLC에서 직접 소켓을 열고 데이터 송수신
- 전용 명령어: OPEN, SEND, RECV, CLOSE

### 소켓 통신 설정
1. PLC 파라미터 > 내장 Ethernet > **오픈 설정**
2. 프로토콜(TCP/UDP), 포트 번호, 상대 IP 설정
3. 프로그램에서 전용 명령어로 송수신

### 프로그램 예 (개념)
```
; 커넥션 오픈
GP.OPEN  Un  S1  D1

; 데이터 송신
GP.SEND  Un  S1  D1

; 데이터 수신
GP.RECV  Un  S1  D1

; 커넥션 클로즈
GP.CLOSE Un  S1  D1
```

## 6. 트러블슈팅

### LED 표시

| LED | 상태 | 의미 |
|-----|------|------|
| OPEN | 점등 | 커넥션 확립 중 |
| SD | 점멸 | 데이터 송신 중 |
| RD | 점멸 | 데이터 수신 중 |
| ERR. | 점등 | 통신 에러 |

### 주요 에러 코드

| 코드 | 원인 | 대책 |
|------|------|------|
| C001H | 상대 기기 무응답 | 케이블/IP 확인 |
| C002H | 타임아웃 | 네트워크 상태 확인 |
| C003H | 커넥션 수 초과 | 불필요한 커넥션 종료 |
| C004H | 포트 번호 중복 | 포트 번호 변경 |
