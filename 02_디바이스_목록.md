# Q시리즈 디바이스 목록

> 출처: Q대응 프로그래밍 매뉴얼(공통명령편)

| 디바이스 | 기호 | 범위 | 설명 |
|----------|------|------|------|
| 입력 | X | X0~X1FFF | 외부 입력 (16진수) |
| 출력 | Y | Y0~Y1FFF | 외부 출력 (16진수) |
| 내부 릴레이 | M | M0~M8191 | 보조 접점 |
| 래치 릴레이 | L | L0~L8191 | 정전 유지 |
| 링크 릴레이 | B | B0~B7FFF | 네트워크용 (16진수) |
| 스텝 릴레이 | S | S0~S8191 | SFC 스텝 |
| 특수 릴레이 | SM | SM0~SM2047 | 시스템 상태 |

## 5. 디바이스 활용 팁

### 래치 설정
- 파라미터 > 디바이스에서 래치(1), 래치(2) 범위 설정
- 래치(1): L.CLR로 클리어 가능
- 래치(2): 리모트 리셋만 가능

### 비트→워드 변환
- **K4M0** = M0~M15를 16비트 워드로 취급
- K1=4비트, K2=8비트, K4=16비트, K8=32비트
