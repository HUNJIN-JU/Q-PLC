## 1. PLC란?

### 정의
- **Programmable Logic Controller**
- 산업 자동화를 위한 디지털 전자 장치
- 릴레이 회로를 **소프트웨어 프로그램**으로 대체

### PLC vs 릴레이 제어

| 항목 | 릴레이 | PLC |
|------|--------|-----|
| 배선 변경 | 실제 배선 수정 | 프로그램만 수정 |
| 공간 | 대형 제어반 | 소형 |
| 유지보수 | 기계적 접점 마모 | 반영구적 |
| 속도 | 수십 ms | 수 ns ~ µs |
| 유연성 | 낮음 | 높음 (타이머, 카운터, 연산 등) |

## 2. PLC 동작 원리

### 스캔 사이클
```
┌─────────────────────────────────────────┐
│  ① 입력 읽기 (INPUT REFRESH)            │
│  ↓                                       │
│  ② 프로그램 실행 (PROGRAM EXECUTION)     │
│  ↓                                       │
│  ③ 출력 쓰기 (OUTPUT REFRESH)            │
│  ↓                                       │
│  ④ 서비스 처리 (통신, 자기진단 등)        │
│  ↓                                       │
│  ①로 돌아감 (반복)                       │
└─────────────────────────────────────────┘
```

### 스캔 타임
- 1사이클에 걸리는 시간 (보통 수 ms ~ 수십 ms)
- 프로그램 크기, I/O 점수에 따라 변동
- **SD203**: 현재 스캔 타임 확인 가능

## 3. 래더 프로그래밍 기초

### 기본 구조
```
 전원모선                                코일
   ┃                                    ( )
   ┠──┤X0├──┤X1├──────────────────(Y0)
   ┃   a접점    a접점               출력
   ┃
   ┠──┤X2├──┬──┤X3├──────────────(Y1)
   ┃        │
   ┃        └──┤X4├───
```

### 자기유지 회로 (가장 기본!)
```
   ┠──┤X0├──┬──┤X1├──────────────(Y0)
   ┃  시작  │  정지(b접점)
   ┃        │
   ┃  ┌─┤Y0├─┘  (자기 유지)
```
- X0(시작 버튼) ON → Y0 ON
- Y0이 자기 접점으로 유지
- X1(정지 버튼) OFF → Y0 OFF

### 인터록 회로
```
   ┠──┤X0├──┤Y1├─(b접점)──────────(Y0)  정회전
   ┃
   ┠──┤X1├──┤Y0├─(b접점)──────────(Y1)  역회전
```
- Y0과 Y1이 동시에 ON되지 않도록 상호 잠금

## 4. 번호 체계 이해

### 10진수 vs 16진수

| 디바이스 | 번호 체계 | 예시 |
|----------|-----------|------|
| X (입력) | 16진수 | X0~XF, X10~X1F |
| Y (출력) | 16진수 | Y0~YF, Y10~Y1F |
| M (내부 릴레이) | 10진수 | M0~M9, M10~M99 |
| D (데이터 레지스터) | 10진수 | D0, D1, D100 |
| T (타이머) | 10진수 | T0, T1, T100 |
| C (카운터) | 10진수 | C0, C1, C100 |

### 상수 표현

| 접두어 | 의미 | 예시 |
|--------|------|------|
| K | 10진 정수 | K100 = 100 |
| H | 16진 정수 | H64 = 100 |
| E | 실수 | E1.5 = 1.5 |

## 5. 실무 기본 패턴

### 패턴 1: ON/OFF 제어
```
LD    X0        ; 시작 버튼
SET   Y0        ; 모터 ON (유지)

LD    X1        ; 정지 버튼
RST   Y0        ; 모터 OFF
```

### 패턴 2: 타이머 지연
```
LD    X0        ; 센서 감지
OUT   T0 K30    ; 3초 후

LD    T0
OUT   Y0        ; 솔레노이드 동작
```

### 패턴 3: 카운터 + 리셋
```
LD    X0        ; 카운트 신호
OUT   C0 K50    ; 50회 카운트

LD    C0
OUT   Y10       ; 50회 도달 시 출력

LD    X10       ; 리셋 버튼
RST   C0        ; 카운터 리셋
```

### 패턴 4: 순차 제어 (스텝 방식)
```
; 스텝 0: 초기 상태
LD    M0
AND   X0
SET   M1        ; 스텝 1로
RST   M0

; 스텝 1: 실린더 전진
LD    M1
OUT   Y0        ; 실린더 A 전진
AND   X10       ; 전진 완료 센서
SET   M2        ; 스텝 2로
RST   M1

; 스텝 2: 클램프
LD    M2
OUT   Y1        ; 클램프 ON
AND   X11       ; 클램프 확인
SET   M3
RST   M2
```
