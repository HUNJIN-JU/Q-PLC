## 3. I/O 할당

### I/O 번호 체계
- **X**: 입력 (16진수 표기) → X0~X1FFF
- **Y**: 출력 (16진수 표기) → Y0~Y1FFF
- 슬롯 위치에 따라 **16점 단위**로 자동 할당

### 기본 할당 예시 (Q35B 베이스)

| 슬롯 | I/O 번호 | 비고 |
|------|---------|------|
| 0 | 전원 유니트 | - |
| 1 | CPU 유니트 | - |
| 2 | X/Y 0000~000F | 1번째 I/O |
| 3 | X/Y 0010~001F | 2번째 I/O |
| 4 | X/Y 0020~002F | 3번째 I/O |

### I/O 유니트 종류

| 모델 | 종류 | 점수 |
|------|------|------|
| QX40 | DC 입력 (싱크) | 16점 |
| QX41 | DC 입력 (소스) | 16점 |
| QX42 | DC 입력 (싱크) | 32점 |
| QY40P | 트랜지스터 출력 (싱크) | 16점 |
| QY41P | 트랜지스터 출력 (소스) | 16점 |
| QY42P | 트랜지스터 출력 (싱크) | 32점 |
| QY10 | 릴레이 출력 | 16점 |

## 4. 배선 방법

### 입력 배선

#### 싱크 입력 (QX40)
```
DC24V(+) ──── 센서 출력 ──── X입력 단자
                              │
DC24V(-) ──── COM 단자 ──────┘
```

#### 소스 입력 (QX41)
```
DC24V(+) ──── COM 단자 ──────┐
                              │
DC24V(-) ──── 센서 출력 ──── X입력 단자
```

### 출력 배선

#### 트랜지스터 출력 (QY40P)
```
DC24V(+) ──── 부하(솔레노이드 등) ──── Y출력 단자
                                        │
DC24V(-) ──── COM 단자 ────────────────┘
```
- **최대 부하 전류**: 0.1A/점 (트랜지스터), 2A/점 (릴레이)
- **서지 흡수 소자**: 유도 부하 시 반드시 설치

## 5. 배터리 관리

### 배터리 종류
- **Q6BAT**: CPU 유니트용 리튬 배터리
- **수명**: 약 5년 (25°C 기준)

### 배터리 교체 시기
- BAT. LED 점등 시 교체
- SM51 (배터리 전압 저하 플래그) ON 시

### 교체 절차
1. **전원 ON 상태**에서 교체 (데이터 보존)
2. 커버 열고 기존 배터리 제거
3. 새 배터리 삽입 후 커넥터 접속
4. GX Works2에서 배터리 에러 이력 클리어

## 6. 접지 및 노이즈 대책

### 접지 방법
- **D종 접지**: 접지 저항 100Ω 이하
- **전용 접지** 권장 (다른 기기와 공용 접지 시 노이즈 유입 가능)

### 노이즈 대책
1. **고압선과 이격**: 제어선은 고압선과 최소 100mm 이격
2. **차폐 케이블**: 아날로그/통신 신호에 사용
3. **서지 킬러**: 유도 부하에 반드시 부착
4. **디지털 필터**: 입력 필터 설정으로 노이즈 제거 (1~60ms)
