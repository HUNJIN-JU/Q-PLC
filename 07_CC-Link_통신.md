## 1. CC-Link 개요

### CC-Link란?
- **Control & Communication Link**
- 미쓰비시 개발 **필드 네트워크** (산업 표준)
- 마스터-슬레이브 구조의 실시간 통신
- PLC, 인버터, 리모트 I/O, 센서 등 연결

### 주요 사양

| 항목 | 사양 |
|------|------|
| 통신 속도 | 156kbps / 625kbps / 2.5Mbps / 5Mbps / 10Mbps |
| 최대 국 수 | 마스터 1국 + 슬레이브 최대 64국 |
| 전송 거리 | 100m(10Mbps) ~ 1200m(156kbps) |
| 케이블 | CC-Link 전용 케이블 (3선식) |
| 토폴로지 | 버스형 (T분기 불가) |

## 4. 디바이스 할당

### CC-Link 전용 디바이스

| 디바이스 | 방향 | 설명 |
|----------|------|------|
| RX | 슬레이브→마스터 | 리모트 입력 (비트) |
| RY | 마스터→슬레이브 | 리모트 출력 (비트) |
| RWr | 슬레이브→마스터 | 리모트 레지스터 읽기 (워드) |
| RWw | 마스터→슬레이브 | 리모트 레지스터 쓰기 (워드) |

### 디바이스 할당 예 (1국당)

| 점유 국수 | RX/RY 점수 | RWr/RWw 워드 |
|-----------|-----------|-------------|
| 1국 | 32점 | 4워드 |
| 2국 | 64점 | 8워드 |
| 3국 | 96점 | 12워드 |
| 4국 | 128점 | 16워드 |

### 리프레시 설정
- 네트워크 파라미터에서 **리프레시 디바이스** 설정
- RX → X, RY → Y, RWr → D, RWw → D 등으로 매핑

## 5. 배선

### CC-Link 케이블 접속
```
[마스터] ──DA──DB──DG──┬── [슬레이브1] ──┬── [슬레이브2] ── ... ──[종단저항]
                       │                │
                    종단저항          중간 접속
```

### 단자 설명

| 단자 | 신호 | 설명 |
|------|------|------|
| DA | Data A | 데이터 신호 A |
| DB | Data B | 데이터 신호 B |
| DG | Data Ground | 신호 그라운드 |
| SLD | Shield | 차폐 접지 |

### 배선 주의사항
- **종단 저항**: 양 끝단에 110Ω 종단 저항 접속
- **케이블 길이**: 총 연장은 통신 속도에 따라 제한
- **분기 불가**: T분기 배선 금지 (데이지 체인만 가능)

## 6. 트러블슈팅

### LED 표시

| LED | 상태 | 의미 |
|-----|------|------|
| RUN | 점등 | 정상 동작 |
| ERR. | 점등 | 에러 발생 |
| L RUN | 점등 | 데이터 링크 실행 중 |
| L ERR. | 점등 | 데이터 링크 에러 |
| SW | 점등 | 스위치 설정 변경 감지 |

### 자주 발생하는 에러

| 증상 | 원인 | 대책 |
|------|------|------|
| L ERR. 점등 | 케이블 단선/접촉 불량 | 케이블 확인 |
| 특정 국 통신 안 됨 | 국 번호 중복 | 국 번호 재확인 |
| 통신 속도 불일치 | 스위치 설정 다름 | 전체 국 속도 통일 |
| 데이터 깨짐 | 노이즈 | 차폐 케이블 사용 |
