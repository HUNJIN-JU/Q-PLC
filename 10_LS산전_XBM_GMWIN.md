## 1. XBM-DN32H 개요

### 제품 사양

| 항목 | 사양 |
|------|------|
| 제조사 | LS산전 (현 LS ELECTRIC) |
| 시리즈 | XBM (XBC 후속) |
| 모델 | XBM-DN32H |
| I/O 점수 | 입력 16점 + 출력 16점 = 32점 |
| 프로그램 용량 | 15K 스텝 |
| 처리 속도 | LD 0.1μs |
| 통신 | RS-232C, RS-485 내장 |
| 전원 | DC24V |

### 디바이스

| 디바이스 | 기호 | 범위 | 설명 |
|----------|------|------|------|
| 입력 | P | P00~P0F | 외부 입력 |
| 출력 | P | P40~P4F | 외부 출력 |
| 보조 릴레이 | M | M0~M2047 | 내부 릴레이 |
| 유지 릴레이 | K | K0~K2047 | 정전 유지 |
| 타이머 | T | T0~T255 | 타이머 |
| 카운터 | C | C0~C255 | 카운터 |
| 데이터 레지스터 | D | D0~D4999 | 16비트 데이터 |
| 특수 릴레이 | F | F0~F127 | 시스템 플래그 |

## 2. GMWIN V4 프로그래밍

### GMWIN 개요
- LS산전 PLC 전용 **프로그래밍 소프트웨어**
- IEC 61131-3 표준 언어 지원
- LD(래더), IL(명령어 리스트), SFC, ST 지원

### 주요 메뉴

| 메뉴 | 기능 |
|------|------|
| 프로젝트 | 새로 만들기, 열기, 저장 |
| 편집 | 래더/IL 편집 |
| 온라인 | 접속, 읽기/쓰기, 모니터 |
| 디버그 | 강제 설정, 모니터링 |

### GMWIN vs GX Works2 비교

| 항목 | GMWIN (LS산전) | GX Works2 (미쓰비시) |
|------|---------------|---------------------|
| 대상 PLC | XGK, XGB, XBM | Q, FX 시리즈 |
| 언어 | LD, IL, SFC, ST | LD, SFC, ST, FBD |
| IEC 표준 | 준수 | 부분 준수 |
| 변수 선언 | 필수 (전역/로컬) | 라벨 또는 직접 지정 |

## 3. GM 주요 명령어

### 기본 명령어

| 명령 | 기능 | 미쓰비시 대응 |
|------|------|--------------|
| LOAD | a접점 | LD |
| LOADN | b접점 | LDI |
| AND | 직렬 a접점 | AND |
| ANDN | 직렬 b접점 | ANI |
| OR | 병렬 a접점 | OR |
| ORN | 병렬 b접점 | ORI |
| OUT | 코일 출력 | OUT |
| SET | 자기유지 ON | SET |
| RST | 자기유지 OFF | RST |

### 데이터 전송

| 명령 | 기능 | 미쓰비시 대응 |
|------|------|--------------|
| MOV | 16비트 전송 | MOV |
| DMOV | 32비트 전송 | DMOV |
| BMOV | 블록 전송 | BMOV |
| FILL | 채움 | FMOV |

### 연산

| 명령 | 기능 | 미쓰비시 대응 |
|------|------|--------------|
| ADD | 덧셈 | + |
| SUB | 뺄셈 | - |
| MUL | 곱셈 | * |
| DIV | 나눗셈 | / |

## 4. 미쓰비시 ↔ LS산전 디바이스 비교

| 기능 | 미쓰비시 Q | LS산전 XBM |
|------|-----------|-----------|
| 입력 | X (16진) | P00~P0F |
| 출력 | Y (16진) | P40~P4F |
| 내부 릴레이 | M | M |
| 유지 릴레이 | L | K |
| 타이머 | T | T |
| 카운터 | C | C |
| 데이터 레지스터 | D | D |
| 특수 릴레이 | SM | F |
| 링크 릴레이 | B | - |

## 5. 주의사항

### 호환성
- 미쓰비시와 LS산전은 **명령어 체계가 다름**
- 프로그램 직접 변환 불가 → 재작성 필요
- 디바이스 번호 체계 다름 (특히 입출력)

### LS산전 특징
- IEC 61131-3 표준 준수도가 높음
- 변수 선언 방식이 더 구조적
- 국내 생산으로 가격 경쟁력
